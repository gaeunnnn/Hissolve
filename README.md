## 서비스 소개

HisSolve는 프로그래밍 경진대회를 위한 온라인 플랫폼으로  
문제 출제, 제출 채점, 실시간 순위 확인 등  
대회 운영에 필요한 기능을 통합 제공하는 서비스입니다.

## 프로젝트 성과

- 약 **100명의 사용자가 참여한 사용자 테스트 진행**
- 실제 경진대회 환경을 가정한 **문제 제출 및 채점 시스템 검증**
- 다수 사용자가 동시에 제출하는 상황에서도 **안정적인 채점 처리 확인**
- Docker 기반 채점 환경으로 **코드 실행 격리 및 보안성 확보**
